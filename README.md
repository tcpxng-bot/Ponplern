# ผ่อนเพลิน 🐷

แอปวางแผนการผ่อนของและเงินเหลือใช้ พร้อมมาสคอตน้องผ่อนเพลิน
ทำงานทั้งหมดในไฟล์เดียว (`index.html`) เก็บข้อมูลในเครื่องด้วย localStorage

## ไฟล์ในชุดนี้
- `index.html` — ตัวแอป (ฝัง React ไว้ในตัว เปิดได้เลย ไม่ต้อง build)
- `manifest.json` — ทำให้ติดตั้งเป็นแอปบนหน้าจอโฮมได้
- `icon.svg` — ไอคอนแอป

## วิธีขึ้น GitHub Pages

1. สร้าง repo ใหม่บน GitHub เช่นชื่อ `pon-ploen`
2. อัปไฟล์ทั้ง 3 ไฟล์เข้าไป (`index.html`, `manifest.json`, `icon.svg`)
   - กดปุ่ม **Add file → Upload files** ลากไฟล์เข้าไปแล้ว Commit
3. ไปที่ **Settings → Pages**
4. ตรง **Branch** เลือก `main` โฟลเดอร์ `/ (root)` แล้วกด **Save**
5. รอสักครู่ จะได้ลิงก์ `https://<ชื่อผู้ใช้>.github.io/pon-ploen/`
6. เปิดลิงก์ด้วย Safari บนไอแพด/ไอโฟน → ปุ่มแชร์ → **เพิ่มไปยังหน้าจอโฮม**

## หมายเหตุ
- ข้อมูล (เงินเดือน, ค่าใช้จ่าย, รายการผ่อน, รหัส) เก็บไว้ในเบราว์เซอร์ของเครื่องนั้น ไม่ได้ขึ้นคลาวด์
  ถ้าล้างข้อมูลเบราว์เซอร์หรือเปลี่ยนเครื่อง ข้อมูลจะไม่ตามไป
- ฟอนต์โหลดจากอินเทอร์เน็ต ถ้าออฟไลน์จะใช้ฟอนต์ระบบแทน (ใช้งานได้ปกติ)
- ลืมรหัส: กด "ลืม?" ที่หน้าใส่รหัสเพื่อตั้งใหม่ (ข้อมูลผ่อนยังอยู่ครบ)
